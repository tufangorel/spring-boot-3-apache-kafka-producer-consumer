## spring-boot-3-apache-kafka-producer-consumer

Purpose : <br/>
1- Send message to Kafka broker topic from kafka-producer-rest-api via kafka-template <br/>
2- Read message from Kafka broker topic to kafka-consumer-rest-api <br/>
